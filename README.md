# 教學管理系統

AI 應用 & n8n 自動化職涯家教 — 1 對 1 線上教學

---

## 目錄結構

```
teaching/
├── clients/
│   └── [個案名稱]/
│       ├── profile.md          ← 個案背景介紹
│       ├── stage1.md           ← 第一階段：模擬諮詢（30 分鐘）
│       ├── stage2.md           ← 第二階段：核心課程（含多次上課紀錄）
│       ├── stage3.md           ← 第三階段：成果諮詢報告
│       └── teaching_plan.md    ← 教學計畫（目標、進度、作業）
└── templates/                  ← 各階段 Markdown 範本
```

## 新增個案流程

1. 在 `clients/` 下建立以個案名稱命名的資料夾
2. 從 `templates/` 複製對應範本，依序重新命名為 `profile.md`、`stage1.md` 等
3. 接案時先填寫 `profile.md` 和 `teaching_plan.md`
4. 每次上課後更新 `stage2.md` 與 `teaching_plan.md`

## 教學三階段

| 階段 | 內容 | 時長 |
|------|------|------|
| 第一階段 | 模擬諮詢，了解背景、確認學習方向 | 30 分鐘 |
| 第二階段 | 核心課程，逐步完成專案 | 每堂 1 小時 |
| 第三階段 | 成果諮詢報告，回顧專案並給出下一步建議 | — |
